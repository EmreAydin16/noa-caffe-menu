# Noa Caffe & Co - QR Menü Sistemi

Bursa, Cumhuriyet Caddesi üzerindeki **Noa Caffe & Co** için dijital QR menü sistemi.

## Kurulum

```bash
npm install
npm start
```

## Kullanım

- **Menü (Müşteri):** http://localhost:3000
- **Yönetim Paneli:** http://localhost:3000/admin

## Özellikler

- Modern ve mobil uyumlu QR menü tasarımı
- Yönetim panelinden ürün ekleme / çıkarma / fiyat değiştirme
- Kategori yönetimi
- QR kod oluşturma ve indirme
- Masa kartı yazdırma şablonu
- Restoran bilgileri düzenleme
